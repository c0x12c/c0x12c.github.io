# c0x12c.github.io

Open source landing page for [Spartan](https://hirespartan.io) at [c0x12c.github.io](https://c0x12c.github.io).

## Stack

- React + Vite
- GitHub Pages + GitHub Actions

## Development

```bash
npm install
npm run dev
```

## Deploy

Push to `master` branch. GitHub Actions builds and deploys automatically.
